# Postman Requests
1. Save Merchant Request CURL
   [POST] /api/merchant/save'
   Request Body-
   {
   "name":"Naman Singh",
   "address":"Haryana, India 121003",
   "mobileNumber":"1234567890",
   "merchantType": "OWNER"
   }
2. Get Merchant By ID
   [GET] /api/merchant/{id}
3. Get All Merchants
   [GET] /api/merchant
4. Delete Merchant By ID
   [DELETE] /api/merchant/{id}
